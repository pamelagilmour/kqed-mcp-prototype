Set up package.json

Add TypeScript configuration

Build a "hello world" MCP server that just connects

Create one tool: search_kqed_posts

Return raw data

Step 4: Enhance the Tool

Add query parameters (date filtering, keywords)
Format responses nicely

Step 5+: Add More Tools

Add /pages search
Add /posts/forum search (for interviews!)
Refine based on usage
